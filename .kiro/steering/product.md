# Product Overview

This is a personal portfolio website for Mr. John Weary, a professional web developer and UI/UX designer. The site showcases his skills, experience, and services through a modern, responsive design.

## Key Features

- Multi-page portfolio (About, Services, Portfolio, Testimonials, Blog, Contact)
- Dark/light theme toggle
- Responsive design for all devices
- Professional branding with orange (#FF5E14) as primary color
- Interactive elements and smooth animations
- Skills showcase with progress bars
- Education and experience timeline
- Service offerings display

## Target Audience

- Potential clients looking for web development services
- Employers seeking experienced developers
- Professional network and industry contacts

## Brand Identity

- Logo: "Codr" with distinctive hamburger menu icon
- Primary color: Vibrant orange (#FF5E14)
- Typography: Poppins font family
- Style: Modern, clean, professional with subtle animations
